# 🌍 AI-Driven Land Use Land Cover Classification (Chennai)

## 📌 Project Overview
This project focuses on Land Use and Land Cover (LULC) classification of Chennai district using satellite imagery and Machine Learning.

## 🧠 Technologies Used
- Google Earth Engine (GEE)
- QGIS
- Sentinel-2 Satellite Data
- Random Forest Algorithm

## 📊 Methodology
1. Data collection from Sentinel-2
2. NDVI calculation
3. Training sample collection
4. Random Forest classification
5. Accuracy assessment
6. Area statistics calculation

## 📈 Results
- Accuracy: **99.74%**
- Kappa: **0.9958**

## 🗺️ Output
- LULC Map (Water, Vegetation, Urban, Barren)
- Area statistics
- Thematic map layout
